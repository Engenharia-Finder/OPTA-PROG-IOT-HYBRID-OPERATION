# OPTA-PROG-IOT-HYBRID-OPERATION
Essa programação tem por objetivo solucionar um problema da Arduino IoT Cloud, onde se o OPTA perder a conexão com a internet, o loop trava.
O tutorial já está no próprio código fonte e também em um arquivo dedicado.


This program aims to solve a problem with the Arduino IoT Cloud, where if the OPTA loses its internet connection, the loop freezes.
The tutorial is already in the source code itself and also in a dedicated file.
